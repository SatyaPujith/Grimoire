# Grimoire Project Guidelines

## Code Style

### TypeScript
- Use strict type checking
- Define interfaces for all data structures
- Prefer `const` over `let`

### React
- Functional components only
- Use hooks for state management
- Props should be typed with interfaces

### CSS/Styling
- Use Tailwind utility classes
- Custom animations in index.html
- Follow naming: `animate-{name}`

## Halloween Theme

### Colors
- **Primary:** Purple (#7c3aed)
- **Accent:** Orange (#f97316)
- **Danger:** Red (#dc2626)
- **Background:** Black (#000000)

### Typography
- **Display:** Nosifer
- **Headings:** Creepster
- **Body:** Crimson Text

### Tone
- Mysterious and spooky
- Educational but entertaining
- Words: crypt, haunted, cursed, forbidden

## Audio Guidelines
- Ambient music: 30% volume
- Sound effects: 80% volume
- Always handle autoplay restrictions

## Deployment
```
VITE_GEMINI_API_KEY=your_api_key
npm run build
```
